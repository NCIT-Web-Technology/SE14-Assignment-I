Introduction

HTML is a markup language used for structing and presenting contents on the world wide web.It was finialized and published on 28 october 2014 by world wide web consortium.This is the fifth version of HTML since the inception of world wide web.

History of HTML5

The web Hypertext application technology working group began work on the new standard in 2004.At that time,HTML 4.02 had not been updated since 2000.IN 2009,W3C allocated the XHTML 2.0 working group's charter to expire and decided not to nenwe it. W3C are currently working together on the development of HTML5.

Features of HTML5

  1. Markup : HTML5 introduces elements and attributes that reflect typical usage on modern websites.
  2. Error loading : HTML5 is designed so that old browsers can safely ignore new HTML constructs.
  3. Popularity : As of 2014,HTML5 is at least partially supported by most popular layout engines.
