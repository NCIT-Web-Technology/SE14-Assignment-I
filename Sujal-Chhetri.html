Defination:

HTTP:

All Web communications transactions use the same protocol- the Hypertext Transfer Protocol(HTTP). HTTP consists of two phases,the request & the response.Each HTTP communication (request&response) between a browser n a Web server consists of two parts,a header & a body.The header contains information about the communication; the body contains the data of the communication.

FTP:

The File transfer Protocol(FTP) is a standard network protocol used to transfer computer files from one host to another host over a TCP-based network,such as the Internet. FTP is built on a client-server architecture & uses separate control & data connections between the client & the server.

SMTP:

Simple Mail Transfer Protocol(SMTP) is an Internet standard for email transmissionn uses TCP port 25 by default;simply used for sending messages.
