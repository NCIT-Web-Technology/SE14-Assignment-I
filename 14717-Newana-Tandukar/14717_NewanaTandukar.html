Prologue

A prologue or prolog is an opening to a story that establishes the setting and gives background details, often some earlier story that ties into the main one, and other miscellaneous information. Web Tech

HTTP
Hypertext Transfer Protocol(HTTP) is a technical specification of a network protocol that provides a standard for Web browsers and servers to communicate.
FTP
File Transfer Protocol(FTP) is the method of transferring data files from one computer to another over a network by dividing files into several segments and assigns a reference number to each one.
SMTP
Simple Mail Transfer Protocol(SMTP) is the standard protocol for email services on a TCP/IP network. SMTP provides the ability to send and receive email messages.
