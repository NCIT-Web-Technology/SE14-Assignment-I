Define the followings:

a.http:

"The hypertext Transfer Protocol(HTTP) is an application protocol for distributed,collaborative,hypermedia information systems.HTTP is foundation of data communication fot the world wide web.Hypertext is structured text that uses logical links (hyperlinks)between nodes containing text".

b.FTP:

"The File Transfer protocol (FTP) is standared netwprk protocol used to transfer computer files from one host to another host over a TCP-based networks,such as the Internet.Ftp is build on a client server architecture and uses separate control and data connection between the client and the server."

c.Smpt:

"Simple Mail Transfer protocol (SMPT) is an internet standared for electronic mail (email) trasmission.First defined by RFC 821 in 1982,it was last updated in 2008 with the Extended SMPT additions by RFC 5321-which is the protocol in widespread use today.SMPT by default uses TCP port 25."
